# Jest Test - Prática em TDD

Repositório retirado do curso da b7web, dedicado à **prática de testes unitários e conceitos de Test-Driven Development (TDD)** usando Jest e TypeScript.

## Objetivo

Consolidar boas práticas em testes, garantir qualidade de código e aprender a escrever código testável desde o início do desenvolvimento.

## Observações

- pasta **lib/** - Possui funções utilitárias com testes unitários.

## Tecnologias

- **Jest** - Framework de testes
- **TypeScript** - Tipagem estática
- **Node.js** - Runtime
- **Sequelize** - ORM para banco de dados

## Como usar

```bash
# Instalar dependências
npm install

# Rodar testes
npm test

# Compilar TypeScript
npm run build
```
